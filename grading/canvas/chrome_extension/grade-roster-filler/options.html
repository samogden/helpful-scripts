Grade Roster Filler — Options

Settings are stored in Chrome and used when you run the popup on a roster page.

Rounding

Rubric

JSON or YAML mapping of cutoffs to letter grades. The key means “if score is < key, use this grade”. Use inf for the final catch-all.

Override Rules (optional)

JSON or YAML array of rules applied after the rubric. Each rule supports either column (string) or columns (list), plus optional agg (min | max | avg, default avg).
